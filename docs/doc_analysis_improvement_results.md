
## 📝 配置选项

### config.yaml 新增配置
```yaml
doc_analysis:
  splitting:
    strategy: "smart"           # smart(智能合并) | fixed(固定拆分)
    min_section_chars: 100      # 最小小节字符数
    merge_related: true         # 是否合并相关小节

  filtering:
    skip_empty: true            # 跳过空小节
    exclude_patterns:           # 排除的小节标题模式
      - "原始数据（JSON）"
      - "Raw Data"
      - "Debug Info"
    min_content_ratio: 0.01     # 最小内容比例（1%）
```
